# Change Request Log

## Nasıl kullanılır
Müşteriden gelen her yeni talep veya kapsam değişikliği buraya kaydedilir.
Prompt engineer değişikliği değerlendirerek geliştirmeye başlama kararını kendisi verir.

## Format
```
### CR-NNN — Kısa başlık
- **Tarih:** YYYY-MM-DD
- **Talep eden:** Müşteri adı / iş birimi
- **Açıklama:** Ne isteniyor?
- **Etkilenen spec bölümleri:** spec.md satır X-Y
- **Etki analizi:** Kaç günlük iş? Hangi modüller etkilenir?
- **Durum:** Beklemede / Onaylandı / Reddedildi
- **Onaylayan:** —
- **ADO Work Item:** #NNN
```
